A simple **Phishing Detection System** built with **Flask**, **HTML**, **CSS**, and **Python**.  
This project helps identify potentially malicious websites by analyzing URLs for suspicious patterns and keywords.

---

## Features

- Detects phishing websites using a rule-based algorithm.
- Provides a **risk score** for each URL.
- Displays detailed **feature analysis**:
  - HTTPS enabled
  - URL length
  - Suspicious keywords
  - IP address usage
- Allows downloading a **scan report**.
- Mobile-friendly and visually appealing UI with gradient backgrounds and soft shadows.

---

## Tech Stack

- **Frontend:** HTML, CSS (responsive and modern design)
- **Backend:** Python, Flask
- **Libraries:** Font Awesome for icons, re (regular expressions) for URL analysis

## How to Run Locally

1. Install dependencies:
    ```bash
    pip install flask
    ```
2. Run the Flask app:
    ```bash
    python app.py
    ```
3. Open your browser and go to:  
   
    http://127.0.0.1:5000/
    


## Usage

- Enter a website URL in the input box.
- Click Analyze Website
- View the risk score and features.
- Optionally, download the **scan report**.
